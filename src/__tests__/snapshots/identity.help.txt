Usage: anima identity|id [options] [command]

Manage agent identities

Options:
  -h, --help             display help for command

Commands:
  create [options]       Create an identity
  list [options]         List identities. Defaults to all orgs you belong to;
                         pass --org or set a default with `am org switch` to
                         filter.
  get [options]          Get an identity by ID
  update [options]       Update an identity
  delete [options]       Delete an identity
  rotate-key [options]   Rotate API key for an identity
  did [options]          Get the DID document for an agent
  credentials [options]  List verifiable credentials for an agent
  card [options]         Get the public agent card
  help [command]         display help for command
