Usage: anima vault [options] [command]

Manage password vault and OAuth authentication

Options:
  -h, --help                       display help for command

Commands:
  provision [options]              Provision vault for an agent
  deprovision [options]            Deprovision vault for an agent
  status [options]                 Check vault status
  sync [options]                   Force vault sync
  store [options]                  Store/create a vault credential
  get [options] <credentialId>     Get credential by ID
  list [options]                   List credentials
  search [options]                 Search credentials
  delete [options] <credentialId>  Delete credential
  generate [options]               Generate password
  totp [options] <credentialId>    Get TOTP code
  share                            Manage credential sharing between agents
  token                            Manage ephemeral vault tokens
  inject [options]                 Detect vault references (vtk_ tokens and
                                   {{vault:...}} templates) in stdin, exchange
                                   them for real credentials, and output the
                                   injected text
  redact [options]                 Redact vault secrets from stdin. Fetches all
                                   credentials for the agent and replaces any
                                   matching secret values with [REDACTED].
  oauth                            Manage OAuth connections for agent
                                   authentication
  help [command]                   display help for command
