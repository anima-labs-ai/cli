Usage: am setup-mcp [options] [command]

Configure MCP server for AI clients

Options:
  -h, --help           display help for command

Commands:
  install [options]    Install Anima MCP server in supported clients
  uninstall [options]  Remove Anima MCP server from supported clients
  status               Show Anima MCP setup status across clients
  verify [options]     Verify Anima MCP configuration for installed clients
  help [command]       display help for command
